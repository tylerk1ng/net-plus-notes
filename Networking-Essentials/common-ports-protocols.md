# Common Ports & Protocols (Sec+ Focus)

| Port | Protocol | Use Case         | Sec+ Threat        |
|------|----------|------------------|--------------------|
| 21   | FTP (TCP)| File transfer    | Cleartext creds   |
| 22   | SSH (TCP)| Remote access    | Brute force       |
| 53   | DNS (UDP)| Resolution       | Amplification     |
| 80   | HTTP     | Web              | XSS               |
| 443  | HTTPS    | Secure web       | MITM              |
| 445  | SMB (TCP)| File share       | Ransomware        |
| 3389 | RDP (TCP)| Remote desktop   | Brute force       |
