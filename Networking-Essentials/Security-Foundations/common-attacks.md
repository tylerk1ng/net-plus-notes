# Common Attacks by OSI Layer (Sec+ Focus)

**L7 Application**: XSS, CSRF, SQL injection (app exploits).

**L6 Presentation**: Phishing, malware (data formatting/encryption exploits).

**L5 Session**: Session hijacking (steals active sessions, cookies/tokens).

**L4 Transport**: SYN flood (DoS), port scanning. 

**L3 Network**: IP spoofing, ICMP redirect.

**L2 Data Link**: ARP poisoning (MITM), MAC flooding.

**L1 Physical**: Sniffing (wiretaps), jamming.

**Study tip**: Sec+ asks "Which layer for session hijacking?" → L5. Mitigation: Secure cookies, timeouts.

**THM**: Session hijacking rooms.
**Messer**: Sec+ Domain 2.2.
