# OSI Model Study Log

This folder is my Network+ OSI model study area based on TryHackMe rooms and Professor Messer videos.

## Goals

- Understand what each OSI layer does and common protocols/devices.
- Connect OSI concepts to real tools and labs from TryHackMe.
- Build quick-review notes and flashcards for exam prep.

## Structure

- `notes/` – Layer-by-layer notes and diagrams.
- `thm/` – Notes from TryHackMe tasks related to the OSI model
- `professor-messer/` – Bullet-point notes.
- `flashcards/` – Questions and answers I might import.

## How I Use This

- Watch a Professor Messer section, pause, and add bullets under the right layer file.
- Do a THM task, then write what I learned in my own words.
- Turn tricky concepts into flashcards inside `flashcards/`.
